## Getting Started
Once you've pulled in the project, navigate to the client folder and enter the following command:
```
{npm run}/yarn install
```
To run the client:
```
{npm run}/yarn start
```
The client will now be running at
```
http://localhost:3000/
```
To build the client:
```
{npm run}/yarn build
```
- Run the node server at localhost:3000
